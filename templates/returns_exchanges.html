Return Policy Details

Eligible Items
  • Unused items in original condition
  • Original packaging with tags
  • All accessories included
  • Within 30 days of delivery
Non-Returnable Items
  • Personalized/custom items
  • Opened software or digital goods
  • Intimate apparel & swimwear
  • Gift cards

Exchange Process

Direct Exchange

Exchange for a different size, color, or variant of the same item.

  • No additional payment if same price
  • Price difference charged/refunded if different
  • Free return shipping for exchanges
  • New item shipped once return is scanned

Return & Repurchase

Return your item and purchase a different item entirely.

  • Return processed normally
  • Refund issued to original payment method
  • Place new order separately
  • Use store credit for immediate purchase
